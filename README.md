# OpenGLWorld
Небольшой C++/OpenGL проект с процедурной генерацией мира, простой физикой и свободной камерой.
## Возможности
- Генерация холмистого мира через фрактальный value-noise.
- Сетка ландшафта с нормалями, освещением и туманом.
- Небо с отдельным шейдером, градиентом, свечением и диском солнца.
- Инвентарь из пяти слотов, подбираемые предметы и hotbar.
- Физика шаров: гравитация, столкновения с землей, столкновения друг с другом и границами мира.
- Камера от первого лица, управление мышью и клавиатурой.
- Переключение паузы и wireframe-режима.
## Сборка
Требования:
- CMake 3.24 или новее.
- Компилятор C++17.
- Драйвер с поддержкой OpenGL 3.3.
Команды:
```powershell
cmake -S . -B build
cmake --build build --config Release
.\build\Release\OpenGLWorld.exe
```
Для Ninja или Makefile путь обычно будет:
```powershell
.\build\OpenGLWorld.exe
```
Быстрая проверка OpenGL-контекста без открытия игрового окна:
```powershell
.\build\Release\OpenGLWorld.exe --smoke
```
## Управление
- `W`, `A`, `S`, `D` - движение камеры.
- `Q`, `E` - вниз и вверх.
- `Shift` - ускорение камеры.
- Мышь - обзор.
- `1`-`5` - выбрать слот инвентаря.
- ЛКМ - бросить выбранный предмет, если он есть в инвентаре.
- `Space` - пауза физики.
- `R` - пересоздать физические тела и предметы.
- `F` - wireframe.
- `Esc` - выход.
