# OpenGLWorld
Небольшой C++/OpenGL проект с процедурной voxel-генерацией, физикой игрока, блоками и инвентарём.
## Возможности
- Генерация блокового мира чанками вокруг игрока с регулируемой дальностью прорисовки.
- Улучшенный ландшафт: холмы, берега, вода, камень, земля, трава, снег, песок, дерево, листья и ресурсы.
- Процедурные текстуры блоков в shader-коде без грубого повторяющегося checker-паттерна.
- Ломание и постановка блоков через raycast от камеры.
- Ломание блоков удержанием ЛКМ с разной прочностью материалов.
- Физика игрока: гравитация, прыжок, sprint, коллизия с блоками и ступеньки.
- Плавание в воде с отдельной скоростью, всплытием и погружением.
- Небо с отдельным шейдером, градиентом, свечением, диском солнца, мягкими процедурными облаками и световыми лучами.
- Прозрачная анимированная вода с несколькими процедурными волнами.
- Освещение от солнца, более мягкие shadow-map тени, процедурные блики, контактные тени и туман.
- Инвентарь с hotbar, 27 слотами, улучшенными иконками, числами стаков, stack/wear-индикаторами, подсказками, сортировкой и перетаскиванием предметов.
- Dear ImGui UI со здоровьем, стаминой, hotbar, инвентарём и текущей дальностью прорисовки.
- Спавн ищет ближайшую землю выше воды.
- Игрок при спавне поднимается из блоков, если генерация заняла точку появления.
- Оптимизация пересборки чанков: новые чанки и сломанные блоки не пересобирают весь мир сразу.
- Переключение wireframe-режима.
## Сборка
Требования:
- CMake 3.24 или новее.
- Компилятор C++17.
- Драйвер с поддержкой OpenGL 3.3.
- Интернет при первой конфигурации CMake, чтобы загрузить GLFW и Dear ImGui через `FetchContent`.
Команды:
```powershell
cmake -S . -B build
cmake --build build --config Release
.\bin\OpenGLWorld.exe
```
Или через готовый батник для Visual Studio solution:
```powershell
.\build_sln.bat
```
Он создаёт `build_vs\OpenGLWorld.sln`, `build_vs\OpenGLWorld.slnx` и `bin\OpenGLWorld.exe`.
Установка инструментов сборки через `winget`:
```powershell
.\install_framework.bat
```
Для Ninja или Makefile путь обычно будет:
```powershell
.\build\OpenGLWorld.exe
```
Быстрая проверка OpenGL-контекста без открытия игрового окна:
```powershell
.\bin\OpenGLWorld.exe --smoke
```
## Управление
- `W`, `A`, `S`, `D` - движение камеры.
- `Space` - прыжок, всплытие в воде.
- `Ctrl` - погружение в воде.
- `Shift` - ускорение.
- Мышь - обзор.
- `1`-`9` - выбрать слот инвентаря.
- Колесо мыши - листать слоты hotbar.
- `Tab` - открыть или закрыть инвентарь.
- `T` - сортировать инвентарь.
- `[` и `]` - уменьшить или увеличить дальность прорисовки.
- Наведение в инвентаре - описание предмета.
- Удерживать ЛКМ в инвентаре - перетащить слот.
- Удерживать ЛКМ - ломать блок.
- ПКМ - поставить выбранный блок.
- `R` - пересоздать мир и сбросить инвентарь.
- `F` - wireframe.
- `Esc` - выход.
